[2019-09-28 Sat]

Binary search is a method for searching an element in an ordered collection of elements.

It works by comparing the element searched for to the one in the middle of the collection. If the element searched for is bigger, reiterate on the subcollection of elements bigger than the middle. If it's smaller, on the subcollection of elements smaller than the middle.
