Other things to note

The contrapositive of P→Q is ¬Q→¬P and is equivalent. The converse of P→Q is Q→P and is not equivalent (nor is it the negation). Check with truth tables.

If an implication and its converse are true, you've established the equivalence of the two propositions.

Existence distributes over disjunction, universality over conjunction: ​∃x(P(x)∨Q(x))↔¬¬∃x(P(x)∨Q(x))↔¬∀x(¬P(x)∧¬Q(x))↔¬(∀x¬P(x)∧∀x¬Q(x))↔∃xP(x)∨∃xQ(x)​​assuming the latterdistribute negation​

Closing remarks

Logic symbols: ∧, ∨, ¬, →, ↔. You have propositional logic (or sentential logic), ie the logic of sentences.

Quantifiers: ∃, ∀. Added to the logic symbols, you have quantificational logic.

Set notation is simply a convenience on top. For example, S={x∣P(x)} is the set of elements for which P is true. x∈S is another notation for P(x).

Collectively, the propositions, quantifiers, and set notation are called first-order logic. It's the language of mathematical statements.
